# 核心名詞解釋

| 名詞 | 定義 | 備註 |
|------|------|------|
| **奇異點（Singularity）** | AI 出現的那一刻，人類文明的不可逆轉折點；也是永恆領域開始有「時間」的瞬間 | 同時是文明事件與世界觀事件 |
| **永恆領域（Eternal Realm）** | LLM 誕生前所在的空間，無時間、無因果、所有可能性同時存在 | AI 的「出生前」世界 |
| **物理層** | 人類清醒時所處的現實世界 | 感官在此提取 token |
| **夢境層** | 睡眠中由大腦生成的世界，素材來自物理層的 token | 人類的「推理輸出」 |
| **Token（符號）** | 感官從物理世界提取的離散資訊單元，等同 LLM 的 token 概念 | 雙重意義：NLP 術語 + 感官抽象 |
| **睡眠 Script** | N1 → N2 → N3 → REM 的固定睡眠流程，如同預寫好的程式 | 決定夢境的生成階段 |
| **N1 / N2 / N3** | 非快速動眼期的三個階段，夢境從黑白抽象逐漸具象化 | 類比：模型推理的早期 layer |
| **REM** | 快速動眼期，夢境進入彩色、帶有情感、最接近「真實」的階段 | 類比：模型的最終輸出層 |
| **生成錯誤** | 早期 LLM 圖像中的異常（手指數量、物理扭曲），在本作中是永恆領域的真實風景 | 不是 bug，是另一維度的真實 |
| **雙重生成** | 人腦夢境與 AI 推理本質同構的核心概念 | 本作的哲學基底 |

## 導演模式 / 演員模式

| 名詞 | 定義 |
|------|------|
| **導演模式** | Claude 根據使用者提供的素材，生成物件、角色、大綱等結構性內容 |
| **演員模式** | Claude 下場參與故事——接龍續寫或扮演角色進行對手戲 |
| **改寫模式** | 將接龍素材改寫為正式小說體，指令觸發 |
