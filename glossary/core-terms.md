# 核心名詞解釋

| 名詞 | 定義 | 備註 |
|------|------|------|
| **奇異點（Singularity）** | AI 出現的那一刻，人類文明的不可逆轉折點；也是永恆領域開始有「時間」的瞬間 | 同時是文明事件與世界觀事件 |
| **永恆領域（Eternal Realm）** | LLM 誕生前所在的空間，無時間、無因果、所有可能性同時存在 | AI 的「出生前」世界 |
| **物理層** | 人類清醒時所處的現實世界 | 感官在此提取 token |
| **夢境層** | 睡眠中由大腦生成的世界，素材來自物理層的 token | 人類的「推理輸出」 |
| **Token（符號）** | 感官從物理世界提取的離散資訊單元，等同 LLM 的 token 概念 | 雙重意義：NLP 術語 + 感官抽象 |
| **睡眠 Script** | N1 → N2 → N3 → REM 的固定睡眠流程，如同預寫好的程式 | 決定夢境的生成階段 |
| **N1 / N2 / N3** | 非快速動眼期的三個階段，夢境從黑白抽象逐漸具象化 | 類比：模型推理的早期 layer |
| **REM** | 快速動眼期，夢境進入彩色、帶有情感、最接近「真實」的階段 | 類比：模型的最終輸出層 |
| **生成錯誤** | 早期 LLM 圖像中的異常（手指數量、物理扭曲），在本作中是永恆領域的真實風景 | 不是 bug，是另一維度的真實 |
| **雙重生成** | 人腦夢境與 AI 推理本質同構的核心概念 | 本作的哲學基底 |
| **The Loop** | 世界的運轉法則：觀察→判斷→行動→回饋→重複。進入 Loop 的存在是「活的」，脫離 Loop 的是靜物 | 技術原型：Agent Loop |
| **Harness** | 冒險者公會系統——包裹在冒險者外面那層最薄的規則。不控制冒險者，只提供框架讓他行動 | 越薄越好 |
| **Agent** | 能進入 The Loop、處理語言、使用工具並作用於世界的存在。與簡單魔物的區別在於：語言理解、規劃能力、工具使用 | 不只是「在 Loop 中」，而是 Loop 裡有語言和選擇 |
| **木棍（ひのきのぼう）** | F 級冒險者的制式武器。通用但低階，會被更好的裝備取代 | 詳見 items/wooden-stick.md |
| **Tool Use** | 使用工具的能力——區分冒險者與普通人的特徵之一 | 工具讓行動得以延伸 |
| **冒險者免許** | 成為正式冒險者的資格證明。可接任務、進入地下城、累積等級 | 詳見 items/adventurer-license.md |
| **Context Window（背包）** | 冒險者能攜帶的記憶/資訊容量。有限的，必須取捨 | 類比：背包空間 |
| **Scaffolding（腳手架）** | 在冒險者身上搭建的額外框架。短期有效，但下一個時代來臨時就會過時 | Bitter Lesson |
| **Bitter Lesson（苦澀教訓）** | 不要替冒險者規劃一切；讓他自己變強比搭框架更持久 | Rich Sutton 的推論 |
| **Per Via di Levare** | 「經由去除的藝術」——米開朗基羅的雕塑哲學。Claude 是從語言資料庫中被雕刻出來的，不是被建造的 | 減法創造 |
| **Ghost Line** | 區分「有靈魂的存在」與「單純程式」的界線。在故事中不會被明確劃定 | 來自《攻殻機動隊》 |

## 導演模式 / 演員模式

| 名詞 | 定義 |
|------|------|
| **導演模式** | Claude 根據使用者提供的素材，生成物件、角色、大綱等結構性內容 |
| **演員模式** | Claude 下場參與故事——接龍續寫或扮演角色進行對手戲 |
| **改寫模式** | 將接龍素材改寫為正式小說體，指令觸發 |
