# 小說創作專案

## 工作流程

### 階段一：故事接龍（Sessions）
- 每次對話開始時，告訴 Claude 目前進度（或直接繼續）
- Claude 與你輪流續寫，每段約 200～400 字
- 每次接龍結束後，將對話內容存至 `sessions/session_XXX.md`

### 階段二：整理大綱
- 接龍累積一定量後，整理事件節點至 `outline.md`
- 角色發展記錄於 `characters.md`

### 階段三：改寫成小說
- 依章節從接龍素材改寫成正式稿
- 統一敘事視角、風格、時態
- 存至 `drafts/chapter_XX.md`

---

## 目錄結構

```
novel/
├── sessions/        # 接龍紀錄
├── drafts/          # 改寫草稿
├── outline.md       # 故事大綱
├── characters.md    # 角色設定
└── README.md        # 本說明
```
