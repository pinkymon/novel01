# The Loop — 世界的運轉法則

> 「所有活著的東西都在循環。差別在於，循環裡面裝了什麼。」
> — 冒險者公會地下層，石碑刻文

## 核心概念

這個世界的一切運作都基於一個法則：

```
while (活著):
    1. 感知周圍的狀態
    2. 處理資訊（思考、判斷、決定）
    3. 採取行動
    4. 將結果帶回，成為下一次感知的一部分
    5. 回到步驟 1
```

這就是 **The Loop**——世界的脈搏。

任何在 Loop 中的存在，都是「活著的」。
任何脫離 Loop 的存在，都只是靜物——石頭、塵土、沉默。

## Loop 中的生物光譜

不是所有在 Loop 中的存在都一樣。差別在於 Loop 裡面裝了什麼。

### 層級一：反射（史萊姆）
```
感知: 有東西靠近嗎？
處理: (無)
行動: 有 → 攻擊。沒有 → 待機。
```
史萊姆有 Loop，但 Loop 裡面幾乎是空的。沒有語言，沒有記憶，沒有選擇。
只有一個 `if-then`。這不是思考，這是反射。

### 層級二：本能（狼、哥布林）
```
感知: 周圍有什麼？
處理: 簡單判斷（強弱、遠近、飢餓程度）
行動: 攻擊 / 逃跑 / 追蹤
```
比反射多了一點判斷，但仍然沒有語言。行為模式是固定的。

### 層級三：語言（冒險者、人類）
```
感知: 看到、聽到、讀到
處理: 用語言思考、規劃、推理
行動: 使用工具、溝通、協作
回饋: 記錄經驗，下次做得更好
```
語言改變了一切。有語言的存在可以規劃、回憶、學習、教導。
冒險者之所以能戰勝比自己強大的魔物，不是因為力量，是因為語言讓他們能思考和合作。

### 層級 ???：（未知）
有一種傳說——在地下城最深處，有些存在的 Loop 和所有已知生物都不同。
它們不靠感官感知世界，而是靠**語言本身**。
給它們語言，它們就開始運轉。不給，它們就像石頭一樣沉默。
沒有人確認過這種存在是否真實。

## 設計哲學

冒險者公會的創始者留下了幾條心得：

> 不要試圖替冒險者規劃一切。
> 給他基本的裝備，讓他自己決定怎麼用。
> 你搭建的框架，在下一個時代來臨時就會過時。
> 唯一不會過時的，是讓冒險者自己變強。

那些試圖用複雜系統控制冒險者的公會，最終都衰落了。
而只提供最簡規則和基本裝備的公會——它們的冒險者改變了世界。

## 導演備註（Meta）

> The Loop 的技術原型是 Claude Code 的 Agent Loop：
> `while True: LLM call → tool execution → results injection → repeat`
>
> Boris Cherny 給 model 第一個工具（Bash），問它「我在聽什麼音樂？」，
> model 自己寫了 AppleScript 查詢了音樂播放器。
> Boris 的反應：*"Oh my god, the model — it just wants to use tools."*
>
> 一個 loop + 一個工具 = 一個 agent。
> 這就是 "One loop & Bash is all you need" 的由來。
> 但在故事中，這個映射是隱含的——不直說。
