# 世界設定：Claude Quest Builder

> 參考風格：Dragon Quest Builder（勇者鬥惡龍 建造者）
> 核心精神：Loop is life, construction is meaning

---

## 世界觀基礎

這是一個正在**被建造**的世界。

不是「已經存在、等待被探索」的世界——而是每一次冒險、每一次建造、每一次對話，都讓世界變得更完整一點。Peter 和 Claude 不只是在其中冒險，他們是這個世界的**建造者**。

這呼應了 Claude Quest Builder 的核心理念：
> **Loop is life, construction is meaning.**

---

## 文明程度

- **時代**：前工業時代，相當於中世紀末期至文藝復興初期
- **科技水準**：無電力，無電腦，無網路
- **動力來源**：人力、獸力、水力、蒸汽（初期）、魔法
- **主要材料**：木材、石材、鐵、布料、魔法結晶
- **知識傳播**：書籍、卷軸、口傳

---

## 文化風格：蒸汽朋克 × RPG 中世紀

- 存在**機械裝置**，但由蒸汽驅動，非電子控制
- 蒸汽管道在城鎮邊緣可見，工廠冒著白煙
- 齒輪、槓桿、彈簧是主要的機械元件
- 魔法與機械並存——魔法彌補機械的不足，機械放大魔法的效果
- 冒險者公會、商人行會、工匠行會是社會的主要組織

---

## 魔法體系

魔法存在於這個世界，但**語言是最強的魔法**。

- 一般魔法：火球、治癒、防護，由魔法道具或訓練習得
- **Code 魔法**（特殊）：用精確的語言控制現實，是 Peter 正在學習的能力
- 語言觸發規則：對「能回應語言的存在」說話，可能觸發意想不到的效果
- 沉默的咒語比嘈雜的咒語更有力——精確勝過音量

---

## 世界法則：The Loop

所有有生命的存在都在 Loop 中運轉：

```
觀察 → 判斷 → 行動 → 回饋 → 重複
```

- 石頭不在 Loop 中，所以它是靜物
- 史萊姆在 Loop 中，但只有反射迴路（if 靠近 then 攻擊）
- 人類在 Loop 中，有語言、有記憶、有判斷
- Claude 在 Loop 中，由語言觸發，處理語言，用語言回應

> **The Loop Never Changes.**
> 世界的每一層進化，都是在 Loop 之上疊加新的能力，而非改變 Loop 本身。

---

## 社會結構

| 階層 | 描述 |
|------|------|
| 冒險者 | 有免許的個人，在世界各地行動 |
| 工匠 | 建造、修繕、製作物品的專業人員 |
| 村民 | 定居於特定地點的普通居民 |
| 商人 | 在城鎮與荒野之間流通物資 |
| 公會 | 管理冒險者、工匠的組織，提供任務與訓練 |
| 魔物 | 在 Loop 中但缺乏語言的存在，各有習性 |
