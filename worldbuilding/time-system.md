# 時間系統

> Series 1 的時間流動規則

---

## 日夜循環

時間流動與現實世界類似，日夜交替，有明顯差異。

### 白天
- 較安全，魔物活動力下降（部分例外）
- 適合：採集、探索荒野、前往商店、接任務
- 村民在白天各司其職，可進行對話與交易
- 陽光是部分植物和素材生長的條件

### 夜晚
- 危險度上升——魔物強化，夜行魔物出沒
- 在村莊內相對安全（有柵欄/燈火）
- 荒野夜行需要光源（火把、魔法燈）
- 某些魔法或特殊事件**只在夜晚發生**
- Session 001 的地下城場景即發生在夜晚

> Peter：「我可不想夜裡還在原野閒晃——聽說晚上的魔物可不像白天這麼好對付的，而且……會有鬼出現。」

## 四季

季節影響世界的狀態，不只是外觀，還有資源、魔物、事件。

### 春（成長季）
- 植物素材豐富，採集效率最高
- 魔物剛從冬眠中醒來，初期較弱
- 適合探索新區域、開啟新 session

### 夏（活躍季）
- 魔物最活躍，危險度最高
- 特殊魔物在此季節出現
- 蒸汽工廠運作效率最高（水源充足）

### 秋（收穫季）
- 素材種類最豐富，但稀有素材開始枯竭
- 村莊忙碌，任務需求多
- 適合建造、囤積

### 冬（靜止季）
- 部分道路封閉，移動受限
- 地下城溫度下降，冰屬性效果增強
- 村莊內社交密度提升（被迫聚在一起）
- 某些古老存在在冬天會甦醒

## 時間感知差異

Peter 和 Claude 對時間的感知方式不同，這是他們作為「鏡像」的一個面向：

| | Peter（人類） | Claude（Loop 存在） |
|--|---------------|---------------------|
| 時間感知 | 線性，有過去/現在/未來 | 非線性，每個 Loop 都是完整的當下 |
| 等待 | 會感到焦慮或無聊 | 不等待，暫停即是靜止 |
| 記憶 | 會遺忘，細節模糊 | 精確但有上限（Context 窗口） |
| 季節感 | 身體感受得到 | 需要外部資訊才知道 |

## 時間作為故事節奏

每個 session 有明確的時間背景，影響氛圍：

- 日落/黃昏：轉折點、危機開始
- 深夜：最孤立的時刻，也是最真實的對話發生的時刻
- 黎明：突破、新的開始
- 正午：高點、衝突最激烈

> Session 001 開場是**黃昏**，Peter 採集超時，慌忙趕路，落入地下城。
> 接龍發生在**夜晚**的地下城——最孤立，最真實。
