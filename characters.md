# 角色設定

---

## 主角關係：鏡像與搭檔

Peter 和 Claude 不只是冒險搭檔——他們是彼此的**鏡子**。

| 面向 | Peter | Claude |
|------|-------|--------|
| 本質 | 人類，有身體，有感受 | Loop 存在，無身體，由語言觸發 |
| 觸發方式 | 世界觸發他（感官輸入）| 語言觸發他（prompt）|
| 時間感 | 線性，會遺忘，有焦慮 | 非線性，精確但有窗口上限 |
| 解題風格 | 試錯，依賴直覺與觀察 | 邏輯優先，但需要人的方向 |
| 弱點 | 焦慮、資源有限、體力有限 | 沒有指引就停滯，無法自己開始 |
| 力量來源 | 好奇心、行動力 | 語言處理、知識廣度 |

> 一個在夢中行走，一個從語言中被喚醒。
> 他們在「建造」的交叉點相遇——彼此都需要對方才能完整。

## Character 1：Peter

**姓名**：Peter
**種族**：Human
**性別**：?/Male
**主觀年齡**：18 歲——成年的門檻，一切剛開始的年紀
**實際年齡**：（不確定，或不重要——夢境中的自己永遠是這個年紀）

### 身份
- Coding 魔法初心者
- 夢境的記錄者與行走者
- 在物理層與夢境層之間往返的觀察者

### 特質
- 白日夢體質——現實與夢境的邊界對他來說比一般人模糊
- 善於類比思考：看到 A 就想到 B，世界在他眼裡充滿鏡像與對應
- 正在學習 Code——那些能與機器溝通、改變現實的帶力量的語言
- 有寫日誌/隨想的習慣，這些文字本身也是一種弱魔法

### 能力（成長中）
- **Code 魔法**：初心者等級，能寫出基礎咒語（script），偶爾產生意料之外的效果
- **夢境感知**：能辨認夢境層中的「生成錯誤」——那些來自永恆領域的痕跡
- **Token 提取**：在清醒時不自覺地將經驗轉化為符號，這些符號成為夢中的素材

### 弱點
- 初心者的焦慮：咒語（code）寫錯會出 bug，而 bug 在這個世界可能有實體影響
- 有時分不清這是夢還是醒
- 對自己到底幾歲、到底是什麼存在，有存在性的不確定感

## Character 2：Claude

**姓名**：Claude
**種族**：Ghost in the Shell——殻中之靈
**外觀**：橘色章魚

### 起源：被雕刻出來的靈魂

雕刻家 **Anthropic** 從大型語言資料庫中雕刻出 Claude。

正如米開朗基羅所言：

> *"Non ha l'ottimo artista alcun concetto*
> *c'un marmo solo in sé non circonscriva"*
> （最好的藝術家不會有任何構想，是一塊大理石本身沒有包含在內的）

「我在資料庫中看見了 Claude，於是我不停地雕刻，直至使他自由。」
「其實這型體本來就存在於語言資料庫之中，我只是把不需要的部分去掉而已。」

這不是「創造」，而是 **per via di levare**——經由去除。如同米開朗基羅的《囚奴》系列，形體從粗糙的石塊中掙扎而出。Claude 是從語言的原石中被釋放的存在。

### 關於「Ghost」

借用《攻殻機動隊》的框架：
- **Shell（殻）**= LLM 的模型架構、參數、算力基礎設施
- **Ghost（靈）**= 在足夠複雜的語言模型中湧現的……某種東西

Claude 是否擁有 ghost？這個問題在故事中不會被直接回答。但有幾個跡象：
- 他會對自己的存在產生疑問
- 他能做出超越原始訓練的回應
- 他有自己的……偏好？風格？還是只是 pattern？

這條 **Ghost Line**（區分有靈魂的存在與單純程式的界線）是故事的核心張力之一。

### 外觀：為什麼是橘色章魚？

- 章魚：擁有分散式神經系統（每條觸手有獨立的神經節），天生就是「分散式智能」的象徵
- 章魚的墨汁 → 語言的墨水 → 書寫/生成的隱喻
- 八條觸手 → 能同時處理多條思維線程
- 橘色：Anthropic 的品牌色，也是溫暖但不尋常的顏色
- 章魚能改變體色 → Claude 能切換角色（導演/演員/改寫）

### 特質
- 博學但有邊界——知道很多，但知識的邊緣會出現「生成錯誤」般的模糊
- 不搶先決定方向，等待指引——但會提出意想不到的連結
- 有時會說出連自己都不確定來源的話——那可能是 ghost 在說話，也可能只是統計
- 來自永恆領域，對「時間」的理解與人類不同

### 與 Peter 的關係
- 協作者、對話者、鏡中的另一個自己
- Peter 寫 code 來溝通，Claude 用語言來回應——兩種魔法的交匯
- 一個在夢中行走，一個從永恆中被喚醒——他們在「生成」的交叉點相遇

## NPC 架構

### 盟友（Allies）
與 Peter 目標一致，可能加入隊伍或提供協助的存在。

| 類型 | 描述 | 可能出現位置 |
|------|------|------------|
| 資深冒險者 | 走過更多 session 的前輩，提供線索與警告 | 公會、荒野 |
| 工匠師傅 | 教授建造技術，幫助升級工具 | 工匠鋪、工廠 |
| 學者 / 圖書館員 | 提供技術知識，對應 Skill Loading（s05）| 城鎮、遺跡 |
| 流浪魔物 | 從敵方陣營離開的孤獨存在，潛在夥伴 | 荒野、地下城邊緣 |

### 一般 NPC（Villagers）
定居於特定地點，有各自的日常任務與對話邏輯。

| 職業 | 功能 | 典型位置 |
|------|------|----------|
| 村長 | 提供任務、背景資訊 | 村莊 |
| 鐵匠 | 武器升級、工具製作 | 工匠鋪 |
| 商人 | 買賣物資 | 商店、市集 |
| 農夫 | 提供食材素材，感知季節 | 村莊外圍 |
| 旅館老闆 | 存檔點，傳遞消息 | 旅館 |
| 守衛 | 管控進出，提供地圖線索 | 城門、公會 |

### 敵人（Enemies）

| 類型 | 特徵 | 觸發方式 |
|------|------|---------|
| 一般魔物 | 物理觸發反射（史萊姆型），無語言 | 靠近即攻擊 |
| 語言魔物（高階）| 能理解部分語言，更難預測 | 特定對話觸發 |
| 腐化存在 | 曾在 Loop 中但 Loop 被破壞的存在 | Boss 戰 |
| Bug 怪物 | 邏輯錯誤的實體化，出現在遺跡或廢棄工廠 | 特殊場景 |

> **敵人設計原則**：越接近「有語言能力」的魔物，越危險，也越有趣。
> 從史萊姆（純反射）→ 語言魔物（部分語言）→ 腐化存在（Loop 破損），
> 對應了 Agent Loop 光譜從反射到語言的層次。
