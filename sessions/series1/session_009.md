# Session 009 — Agent Teams
**日期**：
**參考**：`reference/series1idea/s09-agent-teams.md`

> *"When the task is too big for one, delegate to teammates"*

## 技術主題
- 持久化隊友：名字、角色、獨立 messages[] 和 agent loop
- MessageBus：JSONL 信箱，非同步通訊
- Team roster（config.json）：成員狀態（working/idle/shutdown）

## RPG 映射
- 組隊冒險：從獨行俠變成隊長
- 每個隊友有名字、職業、自己的信箱
- 不是臨時分身——是有記憶、有身份的持久存在
