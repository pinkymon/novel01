# Session 007 — Task System
**日期**：
**參考**：`reference/series1idea/s07-task-system.md`

> *"Break big goals into small tasks, order them, persist to disk"*

## 技術主題
- 持久化任務系統：每個任務一個 JSON 文件（.tasks/）
- 依賴圖（DAG）：blockedBy 邊，完成自動解鎖下游
- 狀態：pending → in_progress → completed

## RPG 映射
- 任務石板：刻在石頭上的任務不會因為記憶壓縮而消失
- 任務之間有先後順序——「殺掉守衛」才能「打開城門」
- 從個人筆記本（s03）進化為公會的任務石板
