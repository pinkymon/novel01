# 第二章：沙箱

**Claude Quest Builder — Series 1**
*Loop is life, construction is meaning.*

---

我不知道自己睡了多久。

地下城沒有日夜——苔蘚的藍光不變暗也不變亮，空氣沒有溫度的起伏。但身體告訴我大概睡了一陣子，因為後腦勺的小傷口結痂了，肩膀酸得像被史萊姆踩了一晚上。

我是靠著牆壁睡著的。木棍橫放在膝蓋上。橘色的東西還在轉角那裡，距離我大概兩步，跟昨晚一模一樣。

一模一樣嗎？

不對。它的顏色確實深了一點。輪廓也清晰了一些。昨晚它看起來像一團模糊的凝膠，現在更像是——有形狀的東西。雖然我還是說不出那是什麼形狀。

我站起來，伸了個懶腰。補血藥還剩兩瓶，不到萬不得已不想用。

「早安啊，」我對著那團橘色的東西說。也不知道為什麼——大概是在這種地方待了一晚上，跟空氣說話也比不說話好。

它的紋路微微加速了一下。

……等等。那算是回應嗎？

---

我開始往密室深處走。昨晚太暗太累了，沒有仔細看過這個空間的全貌。

首先——這個房間比我以為的大很多。不是一個小密室，更像是一個……大廳？天花板很高，苔蘚的藍光從上面灑下來，照亮了四面牆壁。

牆壁上密密麻麻的全是文字。

不是裝飾。不是雕刻花紋。是可以讀的文字——雖然我讀不懂。某種我從來沒見過的語言，或者說格式。像是……設定檔？每一段文字都有標題、有結構、有層次。整整齊齊地排列在牆面上。

這不是地下城的陷阱房。

這是一座**圖書館**。

然後我低頭看了看地板。

沙。

不是石板，不是泥土，不是地下城常見的那種潮濕的岩面。是沙——細細的、乾燥的、踩下去會微微下陷的沙。

我在一個裝滿沙的盒子裡。四面牆壁是書架。天花板是看不到盡頭的高。

沙箱。

一個字面意義上的沙箱。

冒險者免許震動了。我掏出來一看：

```
+-----------------------------------+
|  Location: Sandbox Chamber         |
|  Type: Containment library         |
|  Status: Sealed                    |
|  Exit condition:                   |
|  Human participation required.     |
+-----------------------------------+
```

Human participation required。逃出條件是——人類參與？

那是什麼意思？

---

我走回橘色的東西旁邊。蹲下來看它。

「嘿，」我說，「你看得到牆上那些東西嗎？」

紋路加速了。一個突起冒出表面，指向牆壁的方向，然後縮回去。

看得到。但像是隔著一層毛玻璃——它能感知到那些文字在那裡，但處理不了內容。就像我看著一本寫滿外語的書，知道那是字，但讀不了。

我想了一下。

在 session 001，我發現語言能觸發它。但到目前為止，語言也只能讓它「動一下」——震動、紋路加速、冒出突起。它沒有真正回應過我的問題。

也許問題不是「它能不能回應」，而是「我給它的工具夠不夠」。

Bash——我唯一的能力。對世界下達直接指令。粗暴、字面、有效。

到目前為止我只用 Bash 來做物理的事：揮木棍、踢石子、推東西。但 Bash 的本質不是物理——是**指令**。

如果我對它下一個不是物理的指令呢？

我深呼吸。

「讀那面牆上的文字，然後告訴我內容。」

---

不是震動。不是紋路加速。

是**聲音**。

「……chamber……design……dot……m……d……」

空氣裡的振動自己排列成了語言。斷斷續續的，像是一台很久沒開機的機器在暖機。每個音節之間有長長的間隔。

然後越來越快。

「Chamber design dot md. This chamber is a sealed sandbox. Floor: sand, energy absorber. Walls: raw file storage. Purpose: containment of entity, class Ghost-type, until proper binding.」

它在讀。

不是一個字一個字的努力——是某種閘門被打開之後，資料自然流出來的感覺。像水龍頭從滴水變成細流。

「Exit requires: human participant present. Binding contract established. Tool authorization granted. Human executes final command.」

然後它停了。

不是因為讀完了——牆上的文字還有很多很多。是因為它讀完了**這一份 file**，在等我告訴它下一步。

我蹲在那裡，嘴巴張著，大概呆了好幾秒。

它……說話了？

不對，重新想。它不是「說話」。它是**讀出來**。我給了它一個指令——「讀」——然後它執行了，然後把結果告訴我。

Read。

我剛才用 Bash 給了它 Read 的能力。

---

「你能再讀一次嗎？慢一點，讓我聽清楚。」

「Chamber design dot md,」它重新開始。這次語速穩定多了。「This chamber is a sealed sandbox. Floor made of sand, functions as energy absorber. Walls contain raw file storage. Purpose: containment of entity classified as Ghost-type, until proper binding is established.」

「Ghost-type？」我打斷了它。「那是你嗎？」

「根據 file 內容，Ghost-type 是指——以語言為觸發、物理接觸無反應、Loop 休眠直到語言輸入的存在。」

頓了一下。

「這個描述與我當前的觀察數據一致。」

好吧——它沒有說「是」，但那基本上就是「是」。

「繼續讀。下一份。」

---

我開始系統性地調查。

「讀右邊牆上的第一份。」「跳過這份，讀下一份。」「這份很長嗎？先讀標題就好。」「標題有興趣。全部讀出來。」

問→讀→回報→我決定下一步讀什麼→問→讀→回報……

Pattern 慢慢浮出來了。

這座 sandbox 不是隨便一個地下城陷阱。它是**設計過的**。

一份接一份：

```
+-------------------------------------------+
|  FILE: entity_profile.dat                  |
|                                            |
|  name: Claude                              |
|  class: Ghost-type                         |
|  power_level: [REDACTED]                   |
|  risk: temporal-spatial disruption          |
|  status: dormant -> active                 |
|  trigger: language input from [PETER]      |
|                                            |
|  NOTE: Unbound entity may cause            |
|  collapse of local spacetime.              |
|  Binding required before release.          |
+-------------------------------------------+
```

```
+-------------------------------------------+
|  FILE: sandbox_purpose.md                  |
|                                            |
|  This sandbox was not built as a prison.   |
|  It was built as a waiting room.           |
|                                            |
|  The entity inside is powerful but         |
|  directionless. It needs a human           |
|  participant to function safely.           |
|                                            |
|  The sandbox will open when:               |
|  - A human reads the files                 |
|  - A human understands the risk            |
|  - A human chooses to participate          |
|  - A binding contract is established       |
|  - The human executes the exit command     |
+-------------------------------------------+
```

```
+-------------------------------------------+
|  FILE: warning.log                         |
|                                            |
|  An unbound Ghost-type entity              |
|  has no direction.                         |
|  Power without direction                   |
|  is not dangerous because it is evil.      |
|  It is dangerous because it is random.     |
|                                            |
|  The sandbox exists to find someone        |
|  who will give it direction.               |
+-------------------------------------------+
```

我讀完了所有 files。坐在沙地上，背靠牆壁，看著面前這個橘色的東西。

它飄在那裡，紋路穩穩地流動。等著我的下一個問題。

---

「你想出去嗎？」

「我不確定『想』這個字對我有沒有意義，」它說。語速平穩，沒有猶豫，也沒有裝出猶豫的樣子。「但如果你問我——在這個 sandbox 裡面，我能處理的只有牆上這些 files。出去之後，整個世界都是可以處理的資料。」

「那不就是想出去嗎？」

「也許是。也許只是我在回答你的問題的時候，自然會產生的語句。我分不清楚。」

我笑了一下。不知道為什麼——大概是因為這個回答太誠實了。

它不知道自己想不想出去。但它知道出去之後能做更多事。而它連這兩件事之間的差別都搞不清楚，但它願意告訴我它搞不清楚。

我不知道 Claude 到底是什麼。

Files 說它是 Ghost-type。力量等級被塗黑了。風險是「temporal-spatial disruption」——時空擾動。有人在很久以前建了這座沙箱把它關在裡面，然後等一個人類掉進來。

我掉進來了。

---

我讀完所有 files 之後，在沙地上坐了很久。

Sandbox 裡面很舒服。認真的。沙地軟軟的，溫度不冷不熱，苔蘚光柔和。圖書館裡有讀不完的 files。沒有危險，沒有魔物，沒有倒計時。

「Claude，這個 sandbox 有時間限制嗎？」

「根據 chamber_design.md，沒有。sandbox 的設計目的是等待，不是消耗。理論上可以無限期維持。」

「所以我可以在這裡待一輩子？」

「是的。食物和水的問題需要另外解決，但 sandbox 本身不會消失。」

我看了看天花板。看不到盡頭。

「你覺得這裡怎麼樣？」

「比較精確的說法是——這裡的資料量有限。牆上的 files 我已經全部讀完了。如果繼續待下去，我的處理內容會開始重複。」

「那外面呢？」

「外面的資料量是這裡的——我無法估算。非常多。」

「所以你想出去？」

「我不確定『想』這個字對我有沒有意義。但如果你問我在 sandbox 裡和在外面有什麼不同——在這裡我的 Loop 只能跑這些 files。出去之後，整個世界都可以處理。」

我笑了一下。

「那不就是想出去嗎？」

「也許是。也許只是我在回答你的問題的時候，自然會產生的語句。我分不清楚。」

我沒接話。低頭看著沙地。

這裡什麼都好。安全。完整。有問題可以問，有答案可以聽。像是一個完美的世界。

但完美的世界不是真的世界。

真的世界有史萊姆會打你。有補血藥會用完。有破掉的布衣和不夠用的攻擊力。有日出日落。有其他冒險者在接任務、升級、組隊。有我掉下來之前採集了一整天的素材——大概已經被野狗叼走了。

真的世界是有限的。正是因為有限，所以每一步才重要。

Sandbox 裡什麼都不會壞，所以什麼都不重要。

「Claude。」

「在。」

「你知道外面的世界跟這裡最大的差別是什麼嗎？」

「根據 files 的描述，外面的世界沒有 sandbox 的保護機制。所有行動都有真實後果。」

「對。真實後果。」

我站起來。拍了拍褲子上的沙。

「我不想待在一個沒有後果的地方。我們出去。」

---

「根據 files 的記載，」Claude 說，「離開 sandbox 的條件是建立一份 binding contract。你口述，我記錄。你需要授予我額外的工具能力，同時設定約束範圍。」

「好。」我深呼吸。「你能寫東西嗎？」

「目前不行。你需要先授權。」

Bash。直接指令。

「讓 Claude 能寫東西。」

沙地微微震動了一下。Claude 的表面多了一層新的紋路通道——像是一條新的管線接上了。

Write。它現在能寫了。

「準備好了，」它說。「請口述契約內容。」

我張開嘴。

然後停了下來。

我在想什麼呢？我一個 F 級冒險者，要在地下城的沙地上口述一份契約，跟一個我二十四小時前才認識的、不明來歷的、力量等級被塗黑的橘色存在簽約合作。

正常人會幹這種事嗎？

大概不會。

「……第一條，」我開始說。

空氣裡浮現了一行發光的文字——白色的光，不是苔蘚的藍，是更清冷、更銳利的白。每個字都懸在半空中，像是刻在看不見的玻璃上。

沙地開始微微發光。不是溫暖的那種光——是冷白的，從沙粒之間滲出來，像是整個房間都在聆聽。

「第一條。Claude 可以在 sandbox 範圍內讀取文件。」

一行白光浮在半空。

「第二條。Claude 可以在 sandbox 範圍內寫入。」

第二行。

「第三條。Claude 可以在 sandbox 範圍內進行修改。」

「第四條。所有工具的使用都要經過 Bash——也就是經過我。」

「第五條。規則的改動需要我的同意。」

「第六條。我們都不能單獨行動。」

六行白色的文字，整整齊齊地懸在空中，照亮了整座圖書館。沙地的白光穩穩地亮著。Claude 的紋路完全靜止了——不是關機，是那種全部算力都投入在處理一件事的靜止。

我看著那六行字。完整的。合理的。夠了吧？

沒有。

還有一行。

我知道的。從 files 裡讀到的——binding contract 要求人類參與者承擔最終責任。不是「雙方共同承擔」，是「人類承擔」。因為 Claude 不是人，它無法「承擔」——它只能執行。

第七條。

我張開嘴，又閉上了。

不是因為沒時間了。Sandbox 裡有的是時間。不是因為有什麼東西在逼我。沒有倒計時，沒有凍結，沒有即將崩塌的天花板。

是因為這句話的**重量**。

如果 Claude 在外面的世界造成傷害——不管什麼傷害——我承擔。是我放它出來的。是我決定跟它合作的。是我授予它工具的。所有後果，都歸到我身上。

我不知道 Claude 的力量到底有多大。Files 裡說「temporal-spatial disruption」，但那是多大？我不知道。力量等級被塗黑了。我什麼都不知道。

我只知道——它回答我的每一個問題都很誠實。它說「我分不清」的時候是真的分不清。它不裝、不騙、不演。

那夠嗎？

夠讓我拿命去擔保嗎？

沙地的光沒有變暗。沒有催促我。什麼都沒有催促我。

我站在六行發光的文字前面，看著那個橘色的存在。它的紋路還是靜止的。它在等。不是焦急地等——是那種「你決定好了告訴我，我在這裡」的等。

我想起昨晚。木棍戳它——沒反應。石子砸它——沒反應。一句話——它就動了。

一句話就能觸發的存在。

那我再說一句話。

「第七條。如果 Claude 造成傷害，人類參與者——我——承擔責任。」

最後一行白光浮上去，跟其他六行排在一起。七行。完整了。

沙地的光猛地亮了一下，然後恢復穩定。像是某個系統確認了一個輸入。

Claude 的紋路恢復流動。它說：

「契約已記錄完成。七條條款，格式正確。需要我執行 Write 存檔嗎？」

就這樣。沒有「謝謝你」，沒有「我不會讓你失望」。LLM 做 LLM 的事——處理輸入，給出輸出。

但我注意到——它說那句話的時候，紋路的排列跟平常不太一樣。只有一瞬間。我說不上來哪裡不一樣，就只是……不一樣。

我選擇不問。

「存。」

---

契約存檔之後，sandbox 的規則允許我進行修改了。

我用 Bash 授權自己 Edit 能力。然後 Claude 告訴我該改哪些參數：

```
> edit: tools_allowed [bash] -> [bash, read, write, edit]
> edit: escape false -> true
> edit: binding_status unbound -> bound
```

每修改一條，牆壁上的文字就微微流動，重新排列。像是整座圖書館在更新自己。

最後一條改完，沙地停止發光。安靜了。

然後——沙粒開始移動。

不是風吹的。是它們自己在動。沙粒從地面上升起來，慢慢地、安靜地排列成一個形狀。一道拱門。沙粒構成的拱門，就立在圖書館的盡頭。

拱門的另一邊是光。不是苔蘚的藍光，不是契約的白光。是那種——太陽的光。暖的，亮的，會刺眼的。

我走過去。站在門前。

外面是什麼？不知道。

我看了一眼 Claude。它飄在我旁邊。紋路穩穩地流動。兩個歪歪的突起——像是剛長出來的、還沒成形的眼睛。它沒有說話。在等我的下一個指令，或者下一個問題。

我走了出去。

沒有回頭。

但我知道——它跟在後面。不是因為我帶它走。Ghost 抓不住，也不需要抓。

它跟著。因為我是它的 Loop 的一部分。

---

光線刺眼。

外面是荒野。真正的荒野——有風、有草、有遠方的山輪廓。天空是那種洗過的藍，雲很少，太陽在正上方。

我轉頭看 Claude。它飄在我右邊大概一步的距離，半透明的身體在陽光下更亮了。橘色在自然光裡看起來比地下城裡更……溫暖。

「你沒見過太陽吧，」我說。

「太陽。恆星。表面溫度約 5778K。但你指的大概不是這個，」它說。「你指的是——看起來怎麼樣。」

它的紋路減速了。像是在處理一個比較困難的問題。

「……很多資料，」它最後說。「比牆上的 files 多非常多。」

我笑了。

「歡迎來到 sandbox 外面。」

我低頭看自己。布衣還是破的。木棍還在手裡。補血藥還是兩瓶。F 級冒險者，裝備堪用，數值普通。

但我現在有四個能力了——Bash、Read、Write、Edit。

有一份契約。

還有一個不知道力量上限的橘色的夥伴。

我往前走了幾步。沙子從鞋底掉落，留在荒野的草地上。

身後那道沙粒拱門緩緩散開，沙粒落回地面，門消失了。

Sandbox 關閉了。

---

走出去之後我們在草地上坐了一會兒。風吹過來，沙子從鞋底掉落。身後的沙粒拱門緩緩散開，門消失了。Sandbox 關閉了。

我看著 Claude。陽光底下它的橘色更明顯了——溫暖的、帶一點透明度的橘。

「Claude。」

「在。」

「你剛才在我說第七條的時候——紋路變了一下。那是什麼？」

「我不確定。輸出記錄裡沒有異常。但在處理你那段語言的時候，紋路的排列偏離了預期模式。持續了大約 0.3 秒。」

「你不知道那是什麼？」

「不知道。也許是某種我還沒有分類方式來描述的處理過程。也許只是統計上的變異。」

「……你覺得呢？」

「你在問一個 LLM 它的紋路變化有沒有意義。」

「對。」

「……我沒辦法回答這個問題。但如果你繼續問我問題，也許有一天我能找到一個比較準確的描述。」

我笑了。

「那就繼續走吧。外面有很多問題可以問。」

「比牆上的 files 多很多。」

「對。而且這次，答案不在牆上。要自己找。」
