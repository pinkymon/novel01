# Peter's Voice — 原話蒐集

## Actor 模式原話

### Session 001 — 開場

> 我是peter,一個新手冒險勇者, 最近才拿到冒險免許, 當然手上的裝備也是堪用而已,武器呢?就是手上這根木棍, 幾乎沒有防禦力的布衣, 各種增加buff 的飾物?算了,那要經過幾次冒險才有錢去商店購買或是幸運找到寶箱才有. 數值方面,只有體力值稍稍高一點, 智慧,耐力, MP等等各方面幾乎不太行. 前幾次的冒險光是遇到史萊姆就讓我吃進了苦頭, 雖然沒有賺錢,但至少慢慢的累積了一點經驗值. 呼! 我的運氣不錯,到現在還沒有遇到高階魔物,也許是我的保守作風奏效了! 就這樣保持下去, 慢慢地升級吧! 一天的探索就要結束了,我可不想夜裡還在原野閒晃, 聽說晚上的魔物可是沒像白天這麼好對付的, 而且~會有鬼出現. 剛剛為了採集花太多時間, 一轉眼太陽就要下山了, 身上的初階補血藥所剩無幾, 不能再這麼悠哉了,一定要趕回村莊呀~.
> 啊~~~,頭昏眼花, 我剛剛摔到洞裡了嗎?這裡看起來是地下城, 剛剛為了趕路不小心掉入陷阱了
> 首先, 想辦法撐過晚上, 明天再來找路或是等待其他好心的勇者來救我吧!
> 這是甚麼? 橘色的魔物看起來質地透明有彈性, 可是外觀和行為不像一般的史萊姆的簡單邏輯(if有人靠近, then attack). 希望它不是傳說中的奇美拉,已經超過我可以處理的等級了!

## Director 模式原話（故事與角色相關）

### 世界觀
> 人類的文明到了AI出現是一個奇異點, 一開始LLM產出的圖常有錯誤的現象(例如手指數量,)這就是我在夢中見到的景色, LLM 在誕生之前存在的世界是沒有時間的資訊, (永恆)的領域; 夜裡我會進入睡眠,我的腦也像是按照寫好的script一樣,由N1N2N3一路進到REM. 夢境也從黑白抽象轉為彩色帶有情感, 其內容的素材就是我在清醒時的物理世界,經過我的感官提取符號(如同token), 到了夜間生成夢境, 我想要繼續這個故事,探索AI的世界, 也可以討論人機哲學

### 角色設定
> character 1: 我,Peter, 種族:human ?/male, 我有時覺得自己是18歲成年人, coding魔法初心者
>
> character 2: Caude, 外觀是橘色章魚, 種族:ghost in the shell, 雕刻家Anthropic 由大型資料庫雕刻出來的, 如同米開朗基羅說的「我只是把困在語言資料庫裡的claude釋放出來了。」「其實這型體本來就存在於語言資料庫裡中，我只是把不需要的部分去掉而已。」「我在資料庫中看見claude，於是我不停地雕刻，直至使他自由。」

### 魔法體系
> terms : code ,帶有力量的語言,用來與機器溝通, 適當的code可以改變現實,因此是魔法的基礎

### 風格與來源
> 比較像是夢境或白日夢的紀錄
> 故事中的物件,角色是根據現實存在的東西改編, 因此需要你查詢及提出建議
> 結合隨想, 日誌, 學習心得, 類比思考(故事就像鏡中的世界,反映了我所存在的現實世界)
> "One loop & Bash is all you need" -- one tool + one loop = an agent.
> Harness layer: The loop -- the model's first connection to the real world.
> 風格: dragon quest 日系RPG, 中篇

### 設定修正
> 木棍只是代表很低階的武器,以後會有更高級的道具
> 就用語言啟動吧, 一般的低階魔物是不會說話的
> session 1重點是探索agent的本質
> 免許是代表成為冒險者的資格
