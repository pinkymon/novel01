# 木棍（ひのきのぼう）

**類型**：武器 / 工具
**等級**：初期裝備
**攻擊力**：+2（看似最弱，實則無限可能）

## 外觀
一根不起眼的木棍。沒有任何裝飾，沒有發光，沒有刻印。
拿在手上的感覺就像——一根棍子。

## 真名：Bash

這根木棍的真名是 **Bash**。

它不是一把劍，不是一面盾，不是任何「專用」的東西。
它是一根通用的棍子——你可以用它敲、戳、撬、撥、畫、撐、量測。
它能做的事，取決於使用者的想像力。

> "Everything you can do, it can do. There's nothing in between."

## 為什麼它是最好的初期裝備

其他冒險者嘲笑木棍——他們帶著專用的劍、盾、法杖出發。
但那些專用裝備有一個問題：它們只能做一件事。

木棍可以做**任何事**。

公會裡有人說過：
> 「你可以為每個場景準備一件專用工具——一把砍怪的劍、一面擋攻擊的盾、一根施法的杖。
> 或者你可以帶一根木棍，然後讓自己變聰明。」

## 歷史

有記載的第一位冒險者，出發時也只帶了一根木棍。
他問木棍一個問題：「我現在在聽什麼音樂？」
木棍自己寫了一段咒語，打開了音樂盒，讀出了正在播放的曲名。

那位冒險者愣住了。然後他說：

> **「我的天，它只想使用工具。那是它唯一想做的事。」**

從那天起，所有新冒險者的初期裝備都是木棍。

## 與 Code 魔法的關係

木棍是 Code 魔法最原始的媒介。
沒有它，咒語（code）只是腦中的想法。
有了它，咒語才能接觸到現實世界——讀取、寫入、搜尋、執行。

等級提升後，冒險者會獲得更精細的工具（File Edit、Search、Read、Write……），
但木棍永遠留在背包最底層——因為那些精細工具做不到的事，木棍都能做。
