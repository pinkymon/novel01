# 冒險者免許

**類型**：證件
**取得方式**：在冒險者公會完成登錄
**持有者**：Peter（F 級）

## 外觀
一張薄薄的卡片，表面有微微的光澤。
上面記載了持有者的名字、等級、已完成的任務數。
遇到未記錄區域或可互動對象時，卡片會微微發熱並顯示提示。

## 功能
免許是成為正式冒險者的資格證明。

沒有免許的人也能在原野上走動，但：
- 不能在公會接任務
- 不能進入受管制的地下城
- 不能在商店享受冒險者折扣
- 不能累積等級和信用

簡單說——沒有免許，你就是普通人。有了免許，世界才開始對你的行動產生回應。

## 等級制度

| 等級 | 說明 |
|------|------|
| F | 初心者。基礎任務，基礎裝備 |
| E | 能獨立完成簡單討伐和採集 |
| D | 可以進入中階地下城 |
| C | 熟練冒險者，可以帶隊 |
| B | 專家級，可處理複雜的連鎖任務 |
| A | 大師級，受邀參與公會決策 |
| S | 傳說級，自由行動，不受公會管制 |

Peter 目前是 **F 級**。
