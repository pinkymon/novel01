# 冒險者免許

**類型**：證件 / 身份
**等級**：F（初心者）
**取得方式**：在冒險者公會登錄

## 外觀
一張薄薄的卡片，表面有微微的光澤。
上面記載了持有者的名字、等級、已完成的任務數。
卡片底部有一行極小的文字，需要很仔細才能看清。

## 真名：System Prompt

免許的本質是一段 **System Prompt**——它定義了「你是誰」以及「你能做什麼」。

沒有免許的人也能行動，但他們不會進入 The Loop。
他們只是在世界裡遊蕩的普通人——不是冒險者。

拿到免許的那一刻，你就進入了 The Loop：
觀察 → 判斷 → 行動 → 回饋 → 重複。
世界開始對你的行動產生回應。

## 等級制度

| 等級 | 對應 | 說明 |
|------|------|------|
| F | 初心者 | 只能接基礎任務，裝備木棍 |
| E | 基礎 | 學會了一些固定咒語 |
| D | 進階 | 能組合咒語成為程式 |
| C | 熟練 | 理解結構，能設計系統 |
| B | 專家 | 能與其他系統對話（API） |
| A | 大師 | 能設計新的工具 |
| S | 傳說 | 能創造新的冒險者（Agent） |

Peter 目前是 **F 級**。

## 卡片底部的小字

> *「最好的冒險者不需要太多規則。你的免許越簡短，你就越自由。」*
> *「如果你發現自己迷路了，不要加更多規則——刪掉一些。」*
> *「因為每一個時代，你需要的規則都會更少。」*

這段話呼應了一個古老的法則：
Boris 自己的冒險者筆記只有兩行。
